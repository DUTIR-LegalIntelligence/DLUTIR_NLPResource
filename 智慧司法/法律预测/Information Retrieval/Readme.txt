Legal Document Retrieval using Document Vector Embeddings and Deep Learning 

论文作者的模型结合了法律领域的向量空间表示，其中文档向量生成采用两种不同的机制，同时研究还包含了将法律案例文件表示成不同的向量空间的过程，该过程结合了语义词度量和自然语言处理技术。论文所建立的整合模型，显示出较高的准确度，这证明需要将特定领域的语义相似度纳入信息检索过程，同时也证明针对不同的文档向量维数，不同分布的词的相似度度量可以改进法律信息检索过程。


Legal Question Answering using Ranking SVM and Deep Convolutional Neural Network 

本文是一个竞赛论文，研究了在法律信息提取/蕴涵竞赛中，利用Ranking  SVM 和卷积神经网络完成两项任务：法律信息检索和问答。对于第一个任务，提出的模型使用了三种特征（LSI、Manhattan、Jaccard），并且基于段落级别而不是先前研究中的文章级别。事实上，每一个段落对应于一个多段文章中的一个特定段落。而对于第二个问答任务，信息检索任务的附加统计特征集成到卷积神经网络中有助于提高问答的准确性。
